하지만 이를 markdown으로 렌더링하면 줄바꿈이 되지 않고 다음과 같이 보인다.

this is paragraph. this is sentence 1. this is sentence 2.

this is paragraph.

this is sentence 1.

this is sentence 2.

(여기서의 ·은 space를 의미한다.)

this is paragraph.
